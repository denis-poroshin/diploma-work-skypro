<h2>1. Содержание проекта</h2>
Учебный проект в котором реализована бэкенд часть сайта по перепродаже вещей.
<h2>2. Стек технологий </h2>
    Java
    Maven
    Spring Boot
    Spring Security
    Spring Data
    PostgreSQL
